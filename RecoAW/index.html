Java WikiBooks Recommendation
Content-based recommendation of Java Programming resources for StackOverflow posts - Data was crawled from WikiBooks using Jsoup, and stored in different files after slicing. Slicing was done by separating the content into text and code, and then associating each chunk with its corresponding sub-heading in the page. After tokenizing and filtering the document content, the Analyzer module in Apache Lucene was used to build an index file for all the terms where each term is associated with their respective documents. Searching was performed using the same Analyzer after passing the query string, which returns the top ten recommendations based on similarity to the query.


Additional Implementation
A custom Analyzer was built to replace the StandardAnalyzer of Lucene, to tokenize, filter and stem the documents.
Stemming was performed on the document content using the Porter Stemming algorithm.
Document similarity was determined using BM25 similarity, overriding the default similarity of Lucene.
